Airline Journey EDA Project

Overview
This repository contains a comprehensive data science workflow designed to analyze flight journey patterns and pricing architectures. It processes complex categorical and temporal data to extract actionable business intelligence.

Dataset Features
The analysis utilizes the flights.csv dataset, which includes:

Temporal Data: Journey dates, departure, and arrival times.
Categorical Data: Airline carriers, source/destination cities, and flight routes.
Numerical Data: Pricing metrics and flight durations.
Key Analysis Sections
Data Cleaning: Automated handling of date-time objects and seasonal mapping.
Statistical Profiling: Deep dive into skewness, kurtosis, and outlier detection.
Univariate & Bivariate Analysis: Visualizing price distributions and carrier-specific variances.
Performance Metrics: Identification of busiest routes and top-tier carriers.
Interactive Insights: Dynamic visualizations using Plotly for deep-drill exploration.
Requirements
pandas & numpy for data manipulation.
matplotlib & seaborn for static visualization.
plotly for interactive dashboards.
scipy for statistical testing.
